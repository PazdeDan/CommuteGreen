About this project

This Web site is part of my MYP Personal Project at Asker International School. I started development in November of 2024, with it being complete as of the 6th of February, 2025.

It uses pure HTML, CSS, and JavaScript, together with the HERE Routing API to calculate routes.

I have always had an appreciation for old technology, and I try to design my programs with backwards-compatibility in mind. As such, this Web site is designed to be fully compliant with HTML 4.01 Transitional/Frameset, CSS 2.0, and JavaScript ES3 standards, which all came out in the late 1990s. This, in theory, means that the Web site should be compatible with ancient browsers such as Netscape Navigator 6 or Internet Explorer 5, the latter support systems as old as Windows 3.1 or Mac OS version 7.6.1.

How it works

When you fill in your route details, the JavaScript code grabs all the data from the fields, and uses it to compose the URL for the API request.

Remember, this is not a travel planner! It will only give you a basic summary of the route.

Privacy

My Web site does not share any of your data to third-parties.* The data you see in the historic data table is retrieved from a cookie stored locally on your device, and is not sent to any servers.

*However, Here Technologies, the Web site's API provider, may collect any data you send to them. Please read their Privacy Policy here, before using the API.
